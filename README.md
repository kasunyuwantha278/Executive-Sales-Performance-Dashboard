# Superstore Sales Analysis Dashboard | Power BI

## 📊 Project Overview

This project presents an interactive **Power BI Sales Analytics Dashboard** developed using the Superstore dataset. The objective of this project is to transform raw sales data into meaningful business insights through data cleaning, data modeling, DAX calculations, and interactive visualizations.

The dashboard helps analyze sales performance, profitability, customer segments, product categories, and geographical trends to support data-driven business decisions.

---

## 🎯 Project Objectives

- Analyze overall sales and profit performance
- Identify top-performing products and categories
- Understand customer purchasing patterns
- Analyze regional and country-level sales trends
- Track monthly and yearly sales trends
- Provide interactive business insights using Power BI

---

## 🔄 Data Preparation Process

### Data Cleaning
Performed data preprocessing using Power Query:

- Removed unnecessary columns
- Checked missing values
- Changed data types
- Removed duplicate records
- Created calculated columns

### Data Modeling

Created a star schema model consisting of:

**Fact Table**
- Sales Transactions

**Dimension Tables**
- Date Dimension
- Customer Dimension
- Product Dimension
- Geography Dimension
